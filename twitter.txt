Створити web-app наподобі твітера:

--------------------------------


1) змога реєструватись

2) додавати короткі повідомленя

3) повідомлення можна лайкати, коментувати

4) можна фоловити/анфоловити інших користувачів

5) стрічка повідомлень від зафолованих користувачів

6) список крристувачів онлайн

7) Дані мають бути в JSON файлах: user.json, post.json

8) симуляція роботи REST API для даних з файликів


--------------------------------


Вимоги:

AngularJS >= 1.6

Bootstrap >=
3

Дизайн повинен бути адаптивний під різні дисплеї
